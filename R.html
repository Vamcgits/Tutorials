Introduction to R

R is a powerful programming language and environment specifically designed for statistical computing and data analysis. It is widely used by data scientists, statisticians, and researchers.

Installation

To get started with R, you need to install it on your system. Visit the official R website for installation instructions.

Basic R Syntax

R uses a simple and intuitive syntax. Here are some basic examples: 

            
                # Assigning a variable
                x <- 5

                # Printing a variable
                print(x)

                # Creating a vector
                numbers <- c(1, 2, 3, 4, 5)

                # Calculating the mean
                mean_value <- mean(numbers)
                print(mean_value)

Data Frames

Data frames are a fundamental data structure in R for working with structured data. Here's an example of creating and working with a data frame: 

            
                # Creating a data frame
                data <- data.frame(
                    Name = c("Alice", "Bob", "Charlie"),
                    Age = c(25, 30, 22)
                )

                # Printing the data frame
                print(data)

                # Accessing columns
                print(data$Name)
