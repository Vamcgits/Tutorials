Introduction to MySQL

MySQL is an open-source relational database management system (RDBMS) that is widely used for managing and querying structured data. It is known for its performance, reliability, and scalability.

Installation

Before you start working with MySQL, you need to install it on your system. Visit the official MySQL website for installation instructions.

Connecting to MySQL

You can connect to MySQL using various clients and programming languages. Below is an example of connecting to MySQL using Python: 

            
                import mysql.connector

                # Establish a connection
                connection = mysql.connector.connect(
                    host="localhost",
                    user="your_username",
                    password="your_password",
                    database="your_database"
                )

                # Create a cursor object
                cursor = connection.cursor()

                # Perform database operations
                # ...

                # Close the cursor and connection
                cursor.close()
                connection.close()

Basic SQL Queries

Here are some basic SQL queries to get you started: 

            
                -- Create a new database
                CREATE DATABASE mydatabase;

                -- Use a database
                USE mydatabase;

                -- Create a table
                CREATE TABLE users (
                    id INT AUTO_INCREMENT PRIMARY KEY,
                    username VARCHAR(50),
                    email VARCHAR(100)
                );

                -- Insert data
                INSERT INTO users (username, email) VALUES ('john_doe', 'john@example.com');

                -- Select data
                SELECT * FROM users;

                -- Update data
                UPDATE users SET email='johndoe@example.com' WHERE id=1;

                -- Delete data
                DELETE FROM users WHERE id=1;
